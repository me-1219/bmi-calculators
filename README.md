# BMI-Calculator

A **Body Mass Index (BMI) Calculator** built with **React** and **Tailwind CSS**.  
This app allows users to calculate their BMI based on age, weight, height, and gender.  

## Features
- Supports **Metric (kg, meters)** and **Imperial (lbs, feet)** units
- **Responsive design** for desktop and mobile
- **Light & Dark mode toggle**
- Input validation for age, weight, and height
- Clean and modern UI with Tailwind CSS

## Technologies Used
- React
- Tailwind CSS
- Lucide React icons
